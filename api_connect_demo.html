Step 5 — Mapping tracciato standard
Associa i campi estratti al tracciato aziendale. I campi con * sono obbligatori.
time *
Timestamp evento (epoch o ISO)
hostname *
Host sorgente evento
nomeapp *
Nome applicazione sorgente
tipoazione *
Tipo di azione (login, logout, …)
clientip
IP del client
username
Utente coinvolto
tipooperazione
Tipo operazione specifica
valorePrima
Valore prima della modifica
valoreDP
Valore dopo la modifica
target
Oggetto/risorsa target
note
Note aggiuntive libere
Step 6 — Destinazione output
Index, sourcetype e source dove inviare gli eventi raccolti.
Lo script evita di reinviare eventi già processati.
